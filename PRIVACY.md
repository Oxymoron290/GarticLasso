## Summary

GarticLasso does not collect, store, transmit, or sell any user data. Period.

## What data is accessed

GarticLasso reads only the following information from the page at `garticphone.com`:

- The HTML5 drawing canvas element (to apply lasso fill and tapered brush strokes)
- The currently selected color in Gartic Phone's UI (so the tools use the right color)

This information is used **entirely on your device** and is never sent anywhere.

## What data is collected

**None.** The extension:

- Does not collect personal information
- Does not collect browsing history
- Does not track usage or analytics
- Does not make any network requests
- Does not use cookies or local storage
- Does not transmit any data to external servers

## Permissions

The extension uses a content script that runs on `garticphone.com` only. No other sites are accessed. No special browser permissions are required.

## Contact

If you have questions about this privacy policy, please open an issue on the [GitHub repository](../../issues).
